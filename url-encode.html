Encode atau decode URL-encoded string dengan cepat. Mendukung encodeURIComponent dan encodeURI.

Tentang URL Encode / Decode

URL encoder dan decoder online untuk mengkonversi string URL menjadi format yang aman untuk transmisi dan sebaliknya. URL encoding (juga dikenal sebagai percent-encoding) menggantikan karakter khusus dalam URL dengan representasi persen-heksadesimal (%XX) agar dapat ditransmisikan dengan aman melalui internet. Alat ini penting bagi web developer dan siapa pun yang bekerja dengan URL dan parameter query.

Fitur URL encoder/decoder meliputi: encode teks biasa ke format URL-safe, decode URL encoded string kembali ke teks asli, dukungan penuh untuk karakter Unicode dan multibyte, penanganan spasi sebagai %20 (standar) atau + (application/x-www-form-urlencoded), serta mode encode komponen (hanya mengencode bagian tertentu dari URL). Hasil dapat disalin ke clipboard dengan sekali klik.

URL encoding sangat penting dalam pengembangan web: mengirim data melalui URL parameter GET request, membuat query string untuk API calls, mengencode input form yang dikirim via GET method, membuat bookmarklet, mengencode data untuk redirect URL, dan memastikan URL valid saat berisi karakter khusus seperti spasi, &, ?, #, dan =. Semua pemrosesan dilakukan di browser Anda.
