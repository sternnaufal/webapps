Privacy Policy

Terakhir diperbarui: 21 Juni 2026

1. Informasi yang Kami Kumpulkan

WebApps tidak mengumpulkan data pribadi pengguna. Semua alat berjalan sepenuhnya di sisi klien (client-side) menggunakan JavaScript. Data yang Anda masukkan tidak dikirim ke server mana pun.

2. Cookie dan Teknologi Pelacakan

Kami menggunakan cookie teknis dari layanan pihak ketiga:

  • Bootstrap CDN — untuk memuat stylesheet dan script
  • Google Fonts — untuk rendering tipografi
  • Vercel — platform hosting, dapat mencatat log akses anonim (IP, browser, halaman yang dikunjungi)

Cookie ini bersifat esensial untuk fungsi dasar situs. Kami tidak menggunakan cookie iklan atau pelacakan lintas situs.

3. Data yang Disimpan di Browser

Fitur tema gelap (dark mode) menyimpan preferensi Anda di localStorage browser. Data ini tidak dikirim ke server dan dapat dihapus kapan saja melalui pengaturan browser.

4. Layanan Pihak Ketiga

Situs ini menggunakan layanan berikut yang memiliki kebijakan privasi masing-masing:

  • Vercel Inc. — vercel.com/privacy
  • Google Fonts / Google LLC — policies.google.com/privacy
  • jsDelivr (Bootstrap CDN) — jsdelivr.com/privacy
5. Hak Anda

Anda berhak untuk:

  • Menolak cookie melalui pengaturan browser
  • Menghapus data localStorage kapan saja
  • Mengakses situs secara anonim (tanpa login)
6. Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan akan diumumkan melalui halaman ini.

7. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, hubungi kami melalui halaman Kontak.
