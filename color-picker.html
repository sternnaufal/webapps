Tentang Color Picker

Color picker online untuk memilih dan mengkonversi warna ke berbagai format: HEX, RGB, HSL, dan CSS color name. Dengan color picker tool ini Anda bisa memilih warna melalui visual color spectrum, mengatur hue/saturation/lightness, dan mendapatkan kode warna siap pakai untuk CSS.

Ideal untuk web designer, UI/UX designer, dan developer. Color picker mendukung pemilihan warna secara interaktif, copy kode dengan satu klik, dan history warna yang sudah dipilih. Gratis, tanpa install, langsung dari browser.
